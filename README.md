# Напоминалка Pro - Веб-приложение для управления задачами

## Описание

**Напоминалка Pro** — это клиентское веб-приложение (SPA), разработанное в рамках курсовой работы. Приложение предназначено для создания, управления и отслеживания повседневных задач с функцией оповещения в заданное время. Все данные сохраняются локально в браузере пользователя с использованием LocalStorage API.

### 🎯 Ключевые особенности

-  **Умные уведомления** с предварительными оповещениями
-  **Повторяющиеся задачи** (ежедневно, еженедельно, ежемесячно)
-  **Встроенный календарь** с визуализацией задач
-  **Две темы оформления** (светлая и тёмная)
-  **Категории и приоритеты** для организации задач
-  **Шаблоны задач** с возможностью создания пользовательских
-  **Поддержка ссылок** на внешние ресурсы (онлайн-встречи)
-  **Локальное хранение** данных без серверной части
-  **Экспорт данных** в JSON формат

##  Функционал

### Основные возможности

| Функция | Описание |
|---------|----------|
|  Создание задач | Название, дата/время, описание, ссылка на ресурс |
|  Уведомления | Всплывающие окна + браузерные уведомления + звук |
|  Предварительные оповещения | За 5/15/30/60 минут до события (до 2-х) |
|  Повторение | Ежедневное, еженедельное, ежемесячное с настройкой интервала |
|  Календарь | Месячное представление с цветовыми индикаторами задач |
|  Категории | 6 категорий с цветовой дифференциацией |
|  Приоритеты | 4 уровня: низкий, средний, высокий, срочный |
|  Шаблоны | 4 встроенных + создание пользовательских |
|  Темы | Светлая и тёмная с сохранением выбора |
|  Экспорт | Сохранение всех задач в JSON файл |
|  Отложить | Перенос уведомления на 5 минут |
|  Удаление | Удаление отменённых и выполненных задач |

### Дополнительные возможности

- Фильтрация задач по статусу (все/активные/выполненные)
- Фильтрация по категориям
- Визуальная индикация просроченных задач
- Анимации и микровзаимодействия
- Адаптивный дизайн для мобильных устройств
- Раздел с информацией о разработчике

##  Технологии

### Frontend

| Технология | Назначение |
|------------|------------|
| **HTML5** | Семантическая разметка, формы, модальные окна |
| **CSS3** | Кастомные свойства (переменные), Grid/Flexbox, анимации, медиа-запросы |
| **JavaScript (ES6+)** | Классы, модульная архитектура, обработка событий |

### API и веб-технологии

| Технология | Применение |
|------------|------------|
| **LocalStorage API** | Хранение задач, тем, шаблонов |
| **Notification API** | Браузерные уведомления |
| **Web Audio API** | Звуковые оповещения |
| **Date API** | Манипуляции с датами и временем |

### Архитектурные подходы

- **SPA (Single Page Application)** — одностраничное приложение
- **OOP** — объектно-ориентированное программирование
- **Модульная структура** — разделение на независимые модули
- **Event-driven architecture** — событийно-ориентированное взаимодействие
